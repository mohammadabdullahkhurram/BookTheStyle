# Footage drop-slot

Screen Studio exports land here. Contents are gitignored (media never enters
git); this README holds the slot open and records what the cut expects.

Export settings: MP4 (H.264 or ProRes), 60fps preferred (Remotion retimes to
30), highest quality. Desktop clips at display resolution (2x Retina fine).
Keep Screen Studio's auto-zoom/cursor smoothing ON — that is the point.

Expected clips (names the cut will look for):

| File | Content | Target length (raw) |
| --- | --- | --- |
| `widget-booking.mp4` | Mobile widget: landing → service → stylist → slot → confirm. Portrait. The confirmation tap is the money moment (lands on the 6.34s accent hit). | 15–30s |
| `dashboard.mp4` | Owner dashboard scroll/hover — stat tiles, today list. | 5–10s |
| `calendar-week.mp4` | Master calendar week view — scroll, hover an appointment. | 5–10s |
| `clients.mp4` | Clients directory → open a client profile. | 5–10s |
| `reports.mp4` | Reports page — charts settling, a filter change. | 5–10s |
| `recolor-01.mp4` … `recolor-04.mp4` | Same screen (dashboard or widget) in each accent variant, OR one continuous take switching branding accent in settings. These cross-cut on the drop — frame them identically. | 3–6s each |

Missing clips are reported by the cut, never faked with stills.
